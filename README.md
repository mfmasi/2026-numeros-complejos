Descripción
Se pide crear una clase llamada Complejo que represente un número complejo.

Requisitos
Atributos privados: La clase debe tener dos atributos privados:

reale: parte real del número complejo
imaginario: parte imaginaria del número complejo
Método para mostrar: Implementar una función que retorne el número complejo en formato:

(real, imaginario)
Ejemplo de uso

c = Complejo(3, 4)
print(c.mostrar())  # Salida: (3, 4)
